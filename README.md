## 🚀 Featured Projects

### 🌐 URLytics — Intelligent Domain Intel & Security Auditing
* A specialized tool designed to automate open-source intelligence (OSINT), threat assessment, and domain footprinting.
* **Core Mechanisms:** Engineered multi-layered scripts to query infrastructure properties, aggregate WHOIS histories, parse security headers, and query global registries for threat mapping.
* **Objective:** Streamline target asset mapping and attack surface reductions by piping disparate intelligence data points directly into unified structured payloads.

### 🤖 YARBIZ — Physical AI Assistant Hardware
* Co-developed a physical virtual assistant driven by local AI models and microcontrollers targeted to alleviate workplace stress and optimize administrative overhead.
* **Tech Stack:** Firmware programmed via ESP32 (handling dual-core tasking, I2S audio parsing, and sensor queues) interfacing asynchronously over WebSockets with a Python NLP / TTS inference backend running localized LLMs.
